# 少儿编程平台 (中国版 Scratch)

> 简体中文 | English

一个类似 Scratch 的少儿编程在线平台，采用拖拽式积木块进行可视化编程。

## 功能特性

- **块状编程** - 拖拽式积木块拼搭代码
- **角色与舞台** - 2D 图形、动画、声音支持
- **实时运行** - 项目即时解释执行
- **社区系统** - 作品分享、点赞、评论
- **项目管理** - 创建、编辑、保存、导出

## 技术栈

### 前端

| 技术 | 用途 |
|------|------|
| React 18 + TypeScript | UI 框架 |
| Vite | 构建工具 |
| Blockly | 块状编程引擎 |
| Fabric.js / Konva.js | 2D 画布渲染 |
| Zustand | 状态管理 |
| React Router | 路由管理 |

### 后端

| 技术 | 用途 |
|------|------|
| Node.js + NestJS | API 框架 |
| PostgreSQL | 主数据库 |
| Redis | 缓存/Session |
| MinIO | 对象存储 |

### 部署

| 技术 | 用途 |
|------|------|
| Docker | 容器化 |
| Nginx | 反向代理 |
| 阿里云/腾讯云 | 云服务 |

## 快速开始

### 前置要求

- Node.js 18+
- npm / yarn / pnpm
- Docker (可选)

### 安装

```bash
# 克隆项目
git clone <repo-url>
cd demo

# 安装前端依赖
cd web
npm install

# 安装后端依赖
cd ../server
npm install
```

### 运行

```bash
# 前端开发模式
cd web
npm run dev

# 后端开发模式
cd server
npm run start:dev
```
